# MainChallenge3

A Vite‑based React application that visualizes carbon‑offset calculations.

## Features
- Interactive UI components (`EcoChat`, `OffsetMarket`, `ActivityLogger`, etc.)
- Dockerfile for containerised deployment
- Deployed on Vercel (`https://mainchallenge3.vercel.app/`)

## Setup
```bash
npm install
npm run dev   # start dev server
```

## Deployment
Pushed to GitHub and deployed automatically via Vercel.
